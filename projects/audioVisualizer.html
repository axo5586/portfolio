"Sunset Audio Visualizer"

Solo Project

App

Github

A simple audio visualizer with three embedded mp3s to choose from with a sunset / hill theme in which the user can change the scene and the visualizer itself.

Made with the trio of JS/HTML/CSS, and uses Canvas to draw the scene onto the web page. Audio nodes are taken and manipulated from web page audio and passed through several possible manipulative filters. Display change of the data is done through manipulating the canvas.
